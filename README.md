# Caption Idea

A dual-colored opening screen app that separates Boy and Girl choices for different caption/mood generation.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown in the terminal (usually http://localhost:5173)

## Features

- Dual-colored opening screen (blue for Boy, pink for Girl)
- Split-screen design with clear visual separation
- Interactive hover effects
- Responsive design
